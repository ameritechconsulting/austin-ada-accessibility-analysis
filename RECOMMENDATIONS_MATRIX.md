## 1. PRIORITIZATION FRAMEWORK

### 1.1 Scoring Methodology

Each crosswalk/deficiency is evaluated on multiple dimensions:

#### Risk Score (40% weight)
| Criteria | Points | Definition |
|----------|--------|-----------|
| **Safety Hazard Level** | 0-10 | Trip hazard, structural failure risk, vehicle conflict risk |
| **Population Vulnerability** | 0-10 | % of mobility-impaired residents served by this crossing |
| **Usage Frequency** | 0-10 | Daily pedestrian traffic volume and time-of-use patterns |
| **Environmental Factor** | 0-5 | Weather impacts, seasonal deterioration acceleration |
| **Cumulative Deficiency** | 0-5 | Number of ADA violations at single location |
| **Risk Score = Average × 0.40** | | **Subtotal: 0-10 points** |

#### Equity Score (30% weight)
| Criteria | Points | Definition |
|----------|--------|-----------|
| **Geographic Equity** | 0-10 | Is location in historically underserved area? |
| **School Proximity** | 0-5 | Within 1/4 mile of K-12 school? |
| **Transit Proximity** | 0-5 | Within 500 ft of transit stop? |
| **Healthcare Proximity** | 0-5 | Within 1/2 mile of medical facility? |
| **Senior Population Density** | 0-5 | % of 65+ population in area |
| **Equity Score = Average × 0.30** | | **Subtotal: 0-10 points** |

#### Feasibility Score (20% weight)
| Criteria | Points | Definition |
|----------|--------|-----------|
| **Cost Efficiency** | 0-10 | Cost per beneficiary; leverage opportunities |
| **Implementation Timeline** | 0-10 | Can be completed in <6 months? |
| **Technical Complexity** | 0-10 | No utility conflicts or special design needed? |
| **Funding Availability** | 0-10 | Existing grants or budget allocation match? |
| **Community Support** | 0-5 | Feedback/petition support for improvement |
| **Feasibility Score = Average × 0.20** | | **Subtotal: 0-10 points** |

#### Impact Score (10% weight)
| Criteria | Points | Definition |
|----------|--------|-----------|
| **People Served** | 0-10 | Number of residents gaining accessibility |
| **Visibility/Advocacy** | 0-5 | Media attention, community engagement value |
| **Demonstration Value** | 0-5 | Model project for replication elsewhere |
| **Impact Score = Average × 0.10** | | **Subtotal: 0-10 points** |

### 1.2 Total Priority Score Calculation
```
PRIORITY SCORE = Risk (0-10) + Equity (0-10) + Feasibility (0-10) + Impact (0-10)

Range: 0-40 points
Tier 1 (Critical):     35-40 points → Immediate action (0-3 months)
Tier 2 (High):         28-34 points → Near-term action (3-12 months)
Tier 3 (Medium):       20-27 points → Medium-term (1-2 years)
Tier 4 (Low):          10-19 points → Monitor; include in maintenance cycles
Deferred:              0-9 points  → Reassess with future funding
```
